## How to Host on GitHub Pages

1. Create a new repo: `cf-sg-proxies-page`
2. Upload these 4 files:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
3. Go to **Settings > Pages**
4. Set **Source**: `Deploy from a branch` → `main` → `/ (root)`
5. Save → Wait 1 min → Visit your site!

## How to Test Latency

1. Click **Copy**
2. Set your **browser or system proxy** to that `IP:Port`
3. Click **Check** → See real latency

> Works in **Firefox**, **Chrome (with extension)**, or **system proxy**
